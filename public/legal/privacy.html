Pridebot Privacy Policy

Last Updated: December 11, 2025

1. Data Collection

Pridebot collects and stores the following data:

  • User IDs (Discord User IDs):
    • Command Usage Logging: To track which commands are being used and how frequently, helping us improve bot functionality and user experience.
    • User Profiles: To store and manage custom user profiles, including user-set pronouns, sexual orientation, romantic orientation, gender identity, and optional biography.
    • Pride Avatars: To associate generated pride flag avatars with users who create them.
    • Bot Votes: To record and track votes received from bot listing platforms (Top.gg, DiscordList, etc.) and provide vote rewards.
    • Premium Subscriptions: To track premium membership status and associated benefits for users who subscribe.
  • Server IDs (Discord Guild IDs):
    • Server Configuration: To store server-specific settings and preferences set by server administrators.
    • Command Usage Analytics: To track which servers use which features, helping us understand usage patterns and improve the bot.
    • Server Count: For displaying accurate server statistics on our website and bot listing platforms.
  • User Avatars (Profile Pictures):
    • Pride Avatar Generation: User profile pictures are temporarily processed to create customized pride flag avatars. The original avatar images are not permanently stored beyond the generation process.
  • Custom User Profile Data:
    • Profile Information: User-provided information including pronouns, sexual orientation, romantic orientation, gender identity, biography, and custom profile fields that users voluntarily choose to set.
    • Profile Customization: User preferences for profile display, including colors, themes, and visibility settings.
  • Command Interaction Data:
    • Usage Statistics: General statistics about command usage, including which commands are most popular, response times, and error rates.
    • Error Logs: When commands fail or encounter errors, we log basic information to help us debug and fix issues.
  • Website Analytics:
    • Google Analytics: We use Google Analytics to track website visits, page views, and general usage patterns to improve our website experience. This includes standard web analytics data such as IP addresses (anonymized), browser types, and referral sources.

What We Do NOT Collect:

  • Message content or conversations (we do not read or store your messages)
  • Voice chat data or recordings
  • Direct messages (DMs) sent to the bot are not permanently stored
  • Personal information beyond what you voluntarily provide in your profile
  • Payment information (handled securely by third-party payment processors)
  • Passwords or authentication tokens

2. Use of Data

The collected data is used for the following purposes:

  • Security:
    • Monitoring and ensuring the security of the services by tracking user and server activity to prevent abuse, detect suspicious behavior, and maintain a safe environment.
  • Development:
    • Improving the features and functionality of Pridebot based on user interaction data, including command usage statistics and user feedback, to provide a better user experience.
    • Enhancing profile and avatar features by analyzing how users engage with these tools, allowing for targeted updates and new functionalities.
  • Client Usage:
    • Enabling users to interact with Pridebot effectively, ensuring that commands, features, and services function as intended and meet user needs.
    • Understanding how commands and websites associated with Pridebot are used, enabling the optimization of services and ensuring that they meet user needs.

3. Data Sharing

We do not share your data with third parties except in the following circumstances:

  • When required by law or to respond to legal processes, such as subpoenas or court orders.
  • To protect the rights, property, or safety of our users, Pridebot, or others, including enforcing our Terms of Service and preventing fraud or security breaches.

4. Data Security

We take the security of your data seriously and implement multiple measures to protect it:

  • Database Security: All User IDs, Server IDs, User Profile Information, and bot configuration data are securely stored in a MongoDB encrypted database with authentication and access controls.
  • Access Control: Database access is restricted and only accessible by the bot developer (Sdriver1)
  • Data Encryption: Data is encrypted both in transit (using HTTPS/TLS) and at rest in our database.
  • Limited Retention: We only retain data for as long as necessary to provide our services or as required by law.
  • Regular Security Updates: Our systems and dependencies are regularly updated to patch security vulnerabilities.

However, please note that no security system is impenetrable, and we cannot guarantee absolute security of your data. We continuously review and enhance our security practices to safeguard your information and will notify users in the event of any data breach as required by applicable laws.

5. Data Deletion

You have the right to request deletion of your personal data at any time. We provide three convenient ways to submit a data deletion request:

  • In-Bot Command: Use the /feedback command within Discord and select "Data Deletion Request" to submit your request directly through the bot.
  • Direct Contact: Contact the bot developer Sdriver1 directly via Discord DM with your deletion request.
  • Support Server: Join our Support Server and alert a developer of your request.

Once we receive your deletion request, we will process it promptly and delete your data from our records within 30 days.

6. Changes to the Privacy Policy

We reserve the right to update this Privacy Policy at any time. Your continued use of Pridebot after any changes will constitute your acceptance of the new terms.

7. Contact Us

If you have any questions about this Privacy Policy, please contact Sdriver1 or ask for help in the Support Server.
